Mobile Application


I designed, coded and launched a mobile web application providing access to all online high school resources; it is currently used by 2000+ students, 100+ teachers and 3000+ parents at Fairview Highschool. This is a Ruby on Rails (RoR) web application with MVC framework. RoR runs a Nginx web server and PostgreSQL database and allows for dynamic content by using embed ruby markup. I used RoR in conjunction with jQuery Mobile to design a touch interface. Instead of creating a standalone app each mobile OS, this application is designed to work on any platform. All this was carefully engineered to coexist with school's existing website.

Sample code at: https://github.com/narendasan/mobile-subset On your mobile device go to http://bit.ly/FHSMobile.
