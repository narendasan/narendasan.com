Ekko


Ekko is a Cocoa (OS X) Application in Swfit with a Firebase backend to facilitate elegant and easy to use website sharing. Users can drag windows over to the application to share and collaborate with other remote users. Ekko won the top 10 hacks at Greylock Hackfest 2015 in Burlingame, CA. Code for Ekko is publically available on Github - Ekko.
